State 0 	Load Inputs

State 1		Start/initiate Mont(x,R2)

State 2		Wait for Done1
	    AND shift E to starting bit

State 3		Start Mont(A,XX) 
	    AND Start Mont(A,A) or Mont(XX,XX)

State 4		Wait for Done1 and Done2
			
State 5		Start Mont(A,1)

State 6		Wait for Done1

State 7		Calculation finished
